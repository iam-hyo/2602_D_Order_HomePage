# 🐯 D-Order V2 High-Spec Project

대학 축제 전용 테이블 오더 서비스 **D-Order**의 공식 랜딩 페이지 고도화 버전입니다.

## 📂 폴더 구조 및 Asset 가이드
반드시 `public/assets/` 폴더를 생성하고 아래 이름으로 이미지를 넣어주세요:

1. **메인 페이지**
   - `logo_black.png`, `logo_white.png`: 배경색에 맞춰 변하는 로고
   - `hero_bg_1.jpg`, `hero_bg_2.jpg`: 배경 로테이션
   - `mockup_tablet.png`, `mockup_mobile.png`: 메인 목업
   - `Festival.png`: 연혁표 배경 (30% Dim 자동 적용)

2. **사용 설명서**
   - `manual/manual_1.jpg` ~ `manual_7.jpg`: 카드 클릭 시 상세 이미지

3. **만든 사람들**
   - `makers/미모지_이름.png`: 예) `미모지_전효준.png`

## 🚀 시작하기
1. `npm install`
2. `npm run dev`
